# Stuff

A personal inventory management app built with the help of Agentic AI.

> Note: I built this project to actively explore how Agentic AI can accelerate the software development lifecycle, and to have a practical answer for anyone asking about AI experience on my CV.

Stuff is a lightweight app designed to help you easily catalog and keep track of all your personal belongings.

While built primarily for my own personal use, it is fully containerized and easy to self-host.

## Quick Start

You can pull the image directly from Docker Hub:

```Bash
docker run -d -p 8080:8080 larguma/stuff
```
